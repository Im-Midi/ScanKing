# 美术规范（Clean UI）

设计语言四个词：**干净、圆角、层次、磨砂**。所有界面改动先对照本规范。

## 设计令牌（ui/style.css `:root`）

### 色板

| 令牌 | 浅色 | 深色 | 用途 |
|---|---|---|---|
| --bg | #eef1f6 | #0b0e14 | 页面底 |
| --card | #ffffff | #151a22 | 卡片面 |
| --text | #0f172a | #edf1f7 | 主文字 |
| --text2 | #7c8698 | #8b96a8 | 次要文字 |
| --accent | #2f6bff | #4f8bff | 品牌蓝 |
| --accent-2 | #4f8bff | #6ea3ff | 渐变亮端 |
| --danger | #e5484d | #ff5c61 | 危险操作 |
| --line | rgba(15,23,42,.07) | rgba(255,255,255,.07) | 发丝分割线 |

背景不是纯色：底色上叠两团径向渐变光晕（蓝 7% + 青 6%），页面立刻"有空气"。

### 圆角四档

`--r-sm 12 / --r-md 16 / --r-lg 20 / --r-xl 26`；胶囊元素（标签、chip、toast）用 `999px`。

### 阴影三档（层次的核心）

每档都是"贴地 + 环境光"两层复合：

```css
--shadow-sm: 0 1px 2px rgba(15,23,42,.04), 0 4px 14px rgba(15,23,42,.06);   /* 卡片静置 */
--shadow-md: 0 2px 6px rgba(15,23,42,.05), 0 12px 32px rgba(15,23,42,.10);  /* 悬浮/按下 */
--shadow-lg: 0 8px 20px rgba(15,23,42,.08), 0 24px 64px rgba(15,23,42,.16); /* 弹层 */
```

彩色元素配彩色投影（FAB：`rgba(47,107,255,.35)`），比黑影高级得多。

### 磨砂玻璃配方

```css
background: var(--glass);                    /* 白 72% / 深色 68% */
-webkit-backdrop-filter: blur(20px) saturate(1.7);
backdrop-filter: blur(20px) saturate(1.7);
border-bottom: 1px solid var(--line);        /* 发丝边不可省 */
```

应用位置：顶栏、底栏、编辑器工具条、底部弹层、Toast、相机操作区。**玻璃必须真的悬浮在内容之上**（position:absolute + 滚动区 padding 让内容从玻璃下穿过），否则模糊没有对象，只剩半透明。深色场景用 `--glass-dark`（黑 55-60%）。

## 字体

```
-apple-system, "HarmonyOS Sans SC", "MiSans", "PingFang SC", "Microsoft YaHei", sans-serif
```

层级：页面标题 20/700 · 卡片标题 14/650 · 正文 15/400 · 辅助 12/400 · 按钮 15/650。中文界面克制使用字重，650 是"加粗但不吵"的甜点。

## 图标

全部内联 SVG：`viewBox="0 0 24 24"`，`stroke="currentColor"`，`stroke-width="2"`，圆头圆角（round cap/join），无填充。视觉尺寸 18-28px。新图标必须同风格手绘，禁止混入实心/细线/别家图标库。

## 组件规范

- **文档卡片**：cover 3:4 + 信息区，1px 发丝边 + shadow-sm；按下 scale(0.965) + shadow-md（"抬起来"的错觉）。
- **主按钮**：135° 蓝渐变（accent-2 → accent）+ 彩色投影；次级按钮 accent 10% 底 + accent 文字；危险按钮同构换红。
- **FAB**：62px、圆角 22（squircle 感），按下 scale(0.9) 且 + 号旋转 90°。
- **底部弹层**：顶部 40×4.5px 拖拽把手（::before），圆角 26 只圆上侧，glass-strong + shadow-lg。
- **胶囊角标**：页码用玻璃黑（blur 8px + 白 16% 描边），选中序号用渐变蓝圆。
- **相机**：取景器全屏，操作区玻璃黑压底，快门 74px 白环 + 12% 白光晕。

## 动效

时长 0.13-0.24s。按下反馈一律 scale（0.9-0.965）；视图切换 fadeIn+上移 6px / 0.22s；弹层上滑 `cubic-bezier(0.2, 0.9, 0.3, 1)`（轻微过冲）；加载 spinner 0.8s linear。**不做**长动画、弹跳、视差。

## 品牌

- App 图标：蓝渐变圆角方 + 白纸折角 + 灰文字行 + 青色扫描线横贯（发光）。源文件 `app-icon.png`（1024），由仓库脚本生成各尺寸。
- 青色 #22d3ee 只用于"扫描"隐喻（扫描线、光晕），不做界面主色。
- README 横幅/流程图：`docs/hero.svg`、`docs/flow.svg`，改版时保持同一色板。

## Do / Don't

- ✅ 留白比分割线优先；一屏最多一个渐变主按钮；深浅模式同时验收
- ❌ 纯黑 #000 文字（用 #0f172a）；无边框玻璃；黑色重投影；超过两级的文字灰度；emoji 当图标
